# Zetsukae.com - Official Source Code

This is the official source code repository for [zetsukae.com](https://zetsukae.com), a modern personal blog built with **Astro 5** and the **Koharu** theme.

## Overview

- **Website**: https://zetsukae.com
- **Framework**: [Astro 5](https://astro.build/)
- **Theme**: [Astro Koharu](https://github.com/cosZone/astro-koharu)
- **Deployment**: GitHub Pages
- **Languages**: 🇫🇷 French, 🇬🇧 English, 🇨🇳 Chinese, 🇯🇵 Japanese

## Features

- Modern, responsive design
- Multilingual support (FR, EN, ZH, JA)
- Full-text search with Pagefind
- Dark/Light theme toggle
- Beautiful styled RSS feed
- Optimized performance
- Static site generation
- Mobile-friendly interface

## Technology Stack

- **Astro 5.x** - Modern web framework
- **React** - For interactive components
- **TypeScript** - Type-safe code
- **Tailwind CSS** - Utility-first CSS framework
- **Pagefind** - Client-side search
- **Biome** - Code quality and formatting
